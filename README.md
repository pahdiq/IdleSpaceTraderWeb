# Idle Space Trader Web

Static website files for `gamesdonecorrectly.com`.

## Included files

- `index.html`
- `styles.css`
- `favicon.svg`
- `robots.txt`
- `sitemap.xml`
- `CNAME`

## Publish on GitHub Pages

1. Push this folder to your public repository.
2. In the GitHub repository, open `Settings` -> `Pages`.
3. Under `Build and deployment`, choose `Deploy from a branch`.
4. Select your main branch and the `/ (root)` folder.
5. Save and wait for GitHub Pages to publish the site.

## Domain

The `CNAME` file is already set to:

```text
gamesdonecorrectly.com
```

If you want to use a different domain later, update that file.

## Small customization

The privacy contact email is currently `privacy@gamesdonecorrectly.com` in `index.html`. Change it if you want a different address before publishing.
